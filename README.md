# narrative-engine

A text-based narrative engine written in C, designed for literary, branching storytelling in the spirit of Georges Perec, Marcel Proust, and Jorge Luis Borges

## Build
```bash
make        # build the game
make test   # run tests
make clean  # remove compiled output
```

## Structure

- `src/` — engine source code
- `story/nodes/` — story content files
- `tests/` — test suite
